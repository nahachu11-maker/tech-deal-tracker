API key — one-time setup
Your key is used directly from your browser to api.anthropic.com — it is never sent to this site's server or committed to GitHub. "Remember" stores it only in this browser's local storage; untick on a shared or public computer, and never paste your key anywhere else. Each question costs a fraction of a cent.

Answers are computed by Claude strictly from the tracker's own JSON files, which are approximate and partially covered — so every answer inherits those limits. Treat outputs as a fast first pass, not a verified source; check the underlying deals before quoting numbers anywhere that matters.
